1. Importación

Sube manualmente los PDF a /data en GitHub. Luego presiona el botón para buscar nuevos archivos e importarlos.

Si el sitio está en GitHub Pages, el sistema intenta detectar owner/repositorio automáticamente. En repositorios privados se requiere publicar los PDF o usar manifest.
